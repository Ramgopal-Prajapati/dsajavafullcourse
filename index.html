// Binary Search — O(log n)
public int binarySearch(int[] a, int x) {
  int lo = 0, hi = a.length-1;
  while (lo <= hi) {
    int mid = lo + (hi-lo)/2;
    if (a[mid] == x) return mid;
    if (a[mid] < x) lo = mid+1;
    else hi = mid-1;
  }
  return -1;
}

DS What is DSA?

Definition: Data Structure is a way to organize and store data so it can be used efficiently. Algorithm is a step-by-step procedure to solve a problem. Together (DSA) they decide the speed (Time Complexity) and memory (Space Complexity) of a program.

Real-life Use: Google Maps (Graphs + Dijkstra), Instagram feed (Heaps + Queues), Browser back-button (Stack), Undo/Redo (Stack), Database indexing (B-Trees), Auto-suggestion (Trie).

Operation / ConceptDetail / Complexity
Big-O NotationDescribes worst-case growth: O(1) constant, O(log n) binary search, O(n) linear, O(n log n) merge sort, O(n²) bubble sort.
Why DSACracks FAANG interviews, builds problem-solving brain, writes scalable code.
TypesLinear: Array, LinkedList, Stack, Queue. Non-Linear: Tree, Graph, HashMap, Heap.

Q1. Find Largest Element in Array

Problem: Given int array, return the largest value.

Logic: Initialize max = arr[0]. Loop i=1..n-1, if arr[i]>max → max=arr[i]. Return max. Time O(n), Space O(1).

Java 
public class LargestElement {
    public static void main(String[] args) {
        int[] arr = {12, 45, 7, 89, 34, 56, 91, 23};
        int max = arr[0];
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] > max) max = arr[i];
        }
        System.out.println("Largest element: " + max);
    }
}
Expected Output
Largest element: 91
Live Output




Explanation: Single pass tracking max. Cannot do better than O(n) for unsorted array.


Q2. Reverse an Array (in-place)


Problem: Reverse the array without using extra array.


Logic: Two pointers — left=0, right=n-1. Swap arr[left] & arr[right], move inward until left>=right. O(n) time, O(1) space.




Java





public class ReverseArray {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5, 6};
        int l = 0, r = arr.length - 1;
        while (l < r) {
            int t = arr[l]; arr[l] = arr[r]; arr[r] = t;
            l++; r--;
        }
        for (int x : arr) System.out.print(x + " ");
    }
}


Expected Output
6 5 4 3 2 1 


Live Output




Explanation: Two-pointer technique — used in many problems (palindrome, sort colors).


Q3. Find Second Largest Element


Problem: Find 2nd largest distinct number in array.


Logic: Track largest & secondLargest in single pass. If arr[i]>largest → second=largest, largest=arr[i]. Else if arr[i]<largest && arr[i]>second → second=arr[i].




Java





public class SecondLargest {
    public static void main(String[] args) {
        int[] a = {12, 35, 1, 10, 34, 35};
        int first = Integer.MIN_VALUE, second = Integer.MIN_VALUE;
        for (int x : a) {
            if (x > first) { second = first; first = x; }
            else if (x > second && x < first) second = x;
        }
        System.out.println("Second largest: " + second);
    }
}


Expected Output
Second largest: 34


Live Output




Explanation: Single-pass O(n). Avoid sorting (O(n log n)).

Q5. Binary Search (Sorted Array)


Problem: Find target in sorted array using divide & conquer.


Logic: Maintain low,high. mid=(low+high)/2. If arr[mid]==target → found. If target<arr[mid] → high=mid-1, else low=mid+1. Time O(log n).




Java





public class BinarySearch {
    public static void main(String[] args) {
        int[] arr = {2, 5, 8, 12, 16, 23, 38, 56, 72, 91};
        int target = 23, low = 0, high = arr.length - 1, idx = -1;
        while (low <= high) {
            int mid = low + (high - low) / 2;
            if (arr[mid] == target) { idx = mid; break; }
            else if (arr[mid] < target) low = mid + 1;
            else high = mid - 1;
        }
        System.out.println("Found at index: " + idx);
    }
}


Expected Output
Found at index: 5


Live Output




Explanation: Use low+(high-low)/2 to avoid integer overflow. Array MUST be sorted.

Q8. Rotate Array by K (Reversal Algorithm)


Problem: Rotate array right by k positions.


Logic: k=k%n. Reverse(0..n-1), Reverse(0..k-1), Reverse(k..n-1). O(n) time, O(1) space.




Java





public class RotateArray {
    static void rev(int[] a, int l, int r){while(l<r){int t=a[l];a[l]=a[r];a[r]=t;l++;r--;}}
    public static void main(String[] args) {
        int[] arr = {1,2,3,4,5,6,7};
        int k = 3, n = arr.length; k %= n;
        rev(arr,0,n-1); rev(arr,0,k-1); rev(arr,k,n-1);
        for (int x : arr) System.out.print(x + " ");
    }
}


Expected Output
5 6 7 1 2 3 4 


Live Output




Explanation: Reversal trick — beats O(n*k) brute force.

Q1. Print Matrix in Spiral Order


Problem: Print elements of matrix in spiral order (clockwise).


Logic: Maintain top,bottom,left,right boundaries. Print top row → right col → bottom row → left col, shrink boundaries.




Java





public class SpiralMatrix {
    public static void main(String[] args) {
        int[][] m = {{1,2,3},{4,5,6},{7,8,9}};
        int top=0,bot=m.length-1,lt=0,rt=m[0].length-1;
        while(top<=bot && lt<=rt){
            for(int i=lt;i<=rt;i++) System.out.print(m[top][i]+" ");
            top++;
            for(int i=top;i<=bot;i++) System.out.print(m[i][rt]+" ");
            rt--;
            if(top<=bot){for(int i=rt;i>=lt;i--) System.out.print(m[bot][i]+" ");bot--;}
            if(lt<=rt){for(int i=bot;i>=top;i--) System.out.print(m[i][lt]+" ");lt++;}
        }
    }
}


Expected Output
1 2 3 6 9 8 7 4 5 


Live Output




Explanation: O(r*c). Common matrix interview question.

Q1. Reverse a Linked List


Problem: Reverse a singly linked list iteratively.


Logic: Three pointers: prev=null, curr=head, next. While curr!=null: next=curr.next; curr.next=prev; prev=curr; curr=next. Return prev.




Java





public class ReverseLL {
    static class Node{int v;Node next;Node(int v){this.v=v;}}
    static Node reverse(Node h){
        Node prev=null,curr=h;
        while(curr!=null){Node n=curr.next;curr.next=prev;prev=curr;curr=n;}
        return prev;
    }
    public static void main(String[] args) {
        Node h=new Node(1);h.next=new Node(2);h.next.next=new Node(3);h.next.next.next=new Node(4);
        h=reverse(h);
        for(Node t=h;t!=null;t=t.next) System.out.print(t.v+" ");
    }
}


Expected Output
4 3 2 1 


Live Output




Explanation: O(n) time, O(1) space. Classic interview Q.

Q1. Balanced Parentheses


Problem: Check if string of brackets () [] {} is balanced.


Logic: Push opening brackets. On closing, pop and check matching pair. Empty stack at end = balanced.




Java





import java.util.*;
public class Balanced {
    public static void main(String[] args) {
        String s = "{[()]}";
        Deque<Character> st = new ArrayDeque<>();
        boolean ok = true;
        for(char c : s.toCharArray()){
            if(c=='('||c=='['||c=='{') st.push(c);
            else{
                if(st.isEmpty()){ok=false;break;}
                char t=st.pop();
                if((c==')'&&t!='(')||(c==']'&&t!='[')||(c=='}'&&t!='{')){ok=false;break;}
            }
        }
        System.out.println("Balanced? "+(ok && st.isEmpty()));
    }
}


Expected Output
Balanced? true


Live Output




Explanation: O(n) time & space. Use ArrayDeque (faster than Stack).

Q3. Next Greater Element


Problem: For each element, find next greater element to its right (-1 if none).


Logic: Iterate from right. Use stack — pop elements ≤ current. Top of stack = next greater. Push current.

import java.util.*;
public class NGE {
    public static void main(String[] args) {
        int[] a = {4,5,2,25,7,8};
        int[] res = new int[a.length];
        Deque<Integer> st = new ArrayDeque<>();
        for(int i=a.length-1;i>=0;i--){
            while(!st.isEmpty() && st.peek()<=a[i]) st.pop();

    }
}

Q2. First Non-Repeating Character in Stream


Problem: Given stream of chars, find first non-repeating char so far.


Logic: Use queue + frequency map. Add char to queue; while front has freq>1 → pop. Front = answer.




Java





import java.util.*;
public class FirstNonRepeat {
    public static void main(String[] args) {
        String s = "aabcbc";
        int[] f = new int[26];
        Queue<Character> q = new ArrayDeque<>();
        StringBuilder out = new StringBuilder();
        for(char c : s.toCharArray()){
            f[c-'a']++; q.offer(c);
            while(!q.isEmpty() && f[q.peek()-'a']>1) q.poll();
            out.append(q.isEmpty()? '#' : q.peek()).append(' ');
        }
        System.out.println(out);
    }
}


Expected Output

Live Output




Explanation: O(n) using queue.

public class Fibo {

    static long fib(int n){
        if(n<=1) return n;
        if(memo[n]!=0) return memo[n];
        return memo[n] = fib(n-1)+fib(n-2);
    }
    public static void main(String[] args) {
        System.out.println("fib(10) = "+fib(10));
    }
}

Q2. Merge Sort


Problem: Sort using divide & conquer.


Logic: Divide array in half, sort each half recursively, merge two sorted halves.




Java





import java.util.Arrays;
public class MergeSort {
    static void merge(int[] a,int l,int m,int r){
        int[] L=Arrays.copyOfRange(a,l,m+1), R=Arrays.copyOfRange(a,m+1,r+1);
        int i=0,j=0,k=l;
        while(i<L.length && j<R.length) a[k++] = L[i]<=R[j] ? L[i++] : R[j++];
        while(i<L.length) a[k++]=L[i++]; while(j<R.length) a[k++]=R[j++];
    }
    static void sort(int[] a,int l,int r){
        if(l<r){int m=(l+r)/2; sort(a,l,m); sort(a,m+1,r); merge(a,l,m,r);}
    }
    public static void main(String[] args) {
        int[] a = {38,27,43,3,9,82,10};
        sort(a,0,a.length-1);
        System.out.println(Arrays.toString(a));
    }
}

Q3. Quick Sort

Logic: Pick pivot, partition (smaller left, larger right), recursively sort both halves.

import java.util.Arrays;
public class QuickSort {
    static int part(int[] a,int l,int h){
        int p=a[h], i=l-1;
        for(int j=l;j<h;j++) if(a[j]<=p){i++;int t=a[i];a[i]=a[j];a[j]=t;}
        int t=a[i+1];a[i+1]=a[h];a[h]=t; return i+1;
    }
    static void qs(int[] a,int l,int h){if(l<h){int p=part(a,l,h); qs(a,l,p-1); qs(a,p+1,h);}}
    public static void main(String[] args) {
        int[] a = {10,7,8,9,1,5};
        qs(a,0,a.length-1);
        System.out.println(Arrays.toString(a));
    }
}

Q3. Level Order Traversal (BFS)


Problem: Print tree level-by-level.


Logic: Queue. Push root. While not empty — poll, print, push children.

import java.util.*;
public class LevelOrder {
    static class Node{int v;Node l,r;Node(int v){this.v=v;}}
    public static void main(String[] args) {
        Node root=new Node(1);root.l=new Node(2);root.r=new Node(3);

            if(n.r!=null) q.offer(n.r);
        }
    }
}

Q2. 0/1 Knapsack


Problem: Given weights & values & capacity W, max value (each item once).


Logic: dp[i][w] = max(skip item, take item if fits).




Java





public class Knapsack {
    public static void main(String[] args) {
        int[] wt={1,2,3}, val={6,10,12}; int W=5, n=wt.length;
        int[][] dp = new int[n+1][W+1];
        for(int i=1;i<=n;i++)
            for(int w=0;w<=W;w++){
                dp[i][w] = dp[i-1][w];
                if(wt[i-1]<=w) dp[i][w] = Math.max(dp[i][w], dp[i-1][w-wt[i-1]]+val[i-1]);
            }
        System.out.println("Max value = "+dp[n][W]);
    }
}


Expected Output
Max value = 22


Live Output




Explanation: O(n*W) time. Classic DP problem.
